# XYZ Vapes - 3 Page Website Template

A modern vape website template with **black theme**, **animated canvas smoke background**, **hamburger menu**, and **product card placeholders**. Designed for portfolios, Fiverr showcases, or demo projects.

## Features

- Home page with hero section, floating **Explore** button
- Featured product card placeholders (3 cards)
- Products page with placeholder for 4-5 products
- Disclaimer page for legal/age restrictions
- Top-left shop name, top-right animated hamburger menu
- Canvas smoke background across all pages
- Footer with social icons (WhatsApp, Instagram, Facebook, Twitter, LinkedIn), email, address
- Responsive design for desktop and mobile

## Pages

- `index.html` – Home page  
- `products.html` – Products page  
- `disclaimer.html` – Disclaimer page
